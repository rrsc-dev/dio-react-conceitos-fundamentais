# Instalação do Babel

O Babel é apenas para desenvolvimento

npm install @babel/core @babel/preset-env @babel/preset-react babel-loader -D

## Criar arquivo de configuração do Babel

.babelrc

# Instalação do Webpack

Empacotador de módulos (module bundler)

npm i html-loader html-webpack-plugin webpack webpack-cli webpack-dev-server style-loader css-loader file-loader -D

## Criar arquivo de configuração do Webpack

webpack.config.js

# Adicionar React em uma aplicação que ainda não possui

npm i react react-dom -D
